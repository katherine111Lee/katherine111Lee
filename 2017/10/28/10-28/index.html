如何将伪数组转化成真正的数组

作者:屋塔房里的小孩    发布于: 2017-10-28

什么是伪数组

伪数组从字面意义来看，就是说它不是真正的数组，虽然具有一般数组的length属性，但是却不能调用数组具有的push、pop等方法。这里与大家分享一下伪数组转化成真正数组的三种方法。
假如页面中写入:

var lis=document.querySelectorAll(“li”);
```
此数组具有一般数组的length属性，如下图：

但是如果我们写入：

lis.push(1);
console.log(lis);

会出现以下提示：

所以有时候我们必须要将伪数组转化为真正的数组来满足我们的某些需求。

1. 赋给一个新的空数组

var arr=[];
for(var i=0;i<lis.length;i++){
   arr.push(lis[i]);
   }
   console.log(arr);

2. 利用数组的slice方法

var arr1=Array.prototype.slice.call(lis);
console.log(arr1);
arr1.push(1);
console.log(arr1);

3. 利用es6的from方法

var arr2=Array.from(lis);
console.log(arr2);
arr2.push(2);
console.log(arr2);
